NU FlyEye: Analysis is organized into several complementary submodules:

  1. Data. Organize and query expression data
  2. Processing. Convert measurements to timepoints
  3. Dynamics. Analyze and visualize expression dynamics
  4. Analysis. Detect and quantify spatial expression patterns

Please select a submodule from the sidebar for a detailed descriptions of its implementation.
