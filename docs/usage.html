Example Usage¶

NU FlyEye: Analysis provides a wide range of functionality for analyzing expression data measured using NU FlyEye: Silhouette. A very brief introduction to some core functionalities is provided here. For detailed usage instructions please see the documentation.

Loading an experiment¶

Import an experiment from a directory containing .silhouette files:

#!/usr/bin/env python

from flyeye.data import experiments

path = './data'
experiment = experiments.Experiment(path)
